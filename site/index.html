Citation What is a citation?

What is a citation?

A citation is a quotation from or reference to a book, paper, or author. We cite when we want to:

  • Make reference to a particular work by another author
  • Give credit / Acknowledge contributions made by another author

Example

Let's look at the example on the right. The first thumbnail is a preview of In-line Citations made by an author. The citations appear in the body of a paper, and make reference to other papers. More details about the cited papers would appear in a paper's References section.

There are various styles to display in-line citations. The most common styles are:

  • < author names > [< number >]

    To display authors' name, followed by an identifying number for the References section. This is the style used in our example.

    Example:

    ...in the system described by John Doe [22], it clearly states that...

  • (< author names >, < year of publication >)

    To display authors' name, followed by the year of publication.

    Example:

    ...in the system described by (J. Doe, 2011), it clearly states that...

What is a citing sentence?

You might already have a clue what is a citing sentence, we have already seen one in the examples. The citing sentence is/are the line(s) of words that contain the citation. For example in a part of a paper:

...In this paper, we describe a system that will aid us in taking over the world. Previous works have all identified the key components needed for this task. In the system described by (J. Doe, 2011), it clearly states that...

What is cited information?

When an author cites another paper, his/her purpose must be to either refer to the cited paper as a whole, or refer to a particular piece of information in the cited paper. For example in the paper written by John Doe:

...in order for a system to be successful in taking over the world, there are many components the system must have. Fundamentally, the system must have the ability to.........This enables us to...

As you can see, the cited information is/are the line(s) of words that contain the most relevant information in relation to the citation made by the citing sentence.

Dataset The dataset we use for this project

The dataset we are using for this task is the ACL Anthology Reference Corpus (ACL ARC). ACL ARC is a corpus for scholarly publications in the field of Computational Linguistics. For now, all you need to know about this dataset is that it contains an interlink file.

In the ACL ARC, each scientific paper has an unique ID to identify itself. The example on the right shows a few records in the interlink file.

Let's look at the first record in the example, A00-1003 ==> P99-1029. This shows the citation relationship between 2 papers. It tells us that the scientific paper with the ID A00-1003, is pointing to another paper with the ID P99-1029, that is, the first paper is citing the second paper.

That's about all you need to know about the interlink file for now. Do keep this in mind, as it will be mentioned again in the later part of this page.

Objective What is the objective of this task?

The objective of this task is to gather manual annotations for our research. We wish to annotate whether a citation is General, Specific or Undetermined.

General

A General citation simply means Paper A is citing Paper B in general. Paper A is making reference to Paper B as a whole, or simply giving credit for the work done by authors of Paper B.

Specific

A Specific citation means Paper A is making reference to a specific section of Paper B. For instance, Paper A makes reference to a particular system, System ABC, created/described by Paper B in Paper B's second section (e.g. Our System). Since Paper A is specifically referring to this section of the paper, we say this cite is specific.

Undetermined

In cases where it is not possible to determine the type of the citation.

What is required of this task?

A participant, provided with a set of scientific papers, need to determine the type of citation, and to annotate with details about the citation. To do so, the participant needs to refer to the citing sentence in the citing paper, and attempt to locate the cited information in the cited paper.

How To Annotate? Let's get started!

Recall that the objective of this task to determine the type of the citation, and then to annotate with details about the citation. The challenge of this task to be able to differentiate between General and Specific. Annotating begins with a working file that will be provided to you. Notice that the working file looks really similar to the interlink file. Your task, is to annotate by entering information into this working file. See that the first 3 lines has already been filled with the details. Don't worry about that dash / numbers / question mark means for now, we'll explain it shortly.

The general approach to annotating is as follow:

if determinable
	if specific # Able to find a specific location for cited information
		if single-location # Cited info resides within 30 lines
			if multi-line # Cited info resides in a range of lines
				Annotate with range, e.g. 65-80
			else if single-line # Cited info resides in a line
				Annotate with number, e.g. 65
		else if multi-locations # Cited info resides at different locations in cited paper
			if multi-line
				Annotate with range + '!', e.g. 65-80!143-160!...
			else if single-line
				Annotate with number + '!', e.g. 65!150!...
	else
		Annotate with '-'
else
	Annotate with '?'

# Note: For multi-locations, it can be a combination of single-line and multi-line i.e. 46!153-188 or 153-188!200

Here's how

In the working file, we capture annotations in "2 columns" separated by the comma.

< cite >,< details >

General A single dash (or minus sign)

A General citation is annotated with a single "dash". For example:

A00-1003==>P99-1029,-

We annotate a cite as General when we cannot find a specific location for a cited information.

Undetermined A single question mark

A Undetermined citation is annotated with a single "question mark". For example:

A00-1004==>J93-1006,?

Example (in the cited paper):

189
190
191
based tagger by inferring rules...

The numbers you can see in this examples are actually page numbers in the original paper. So instead of having the actual content, you only have the page number, which is definitely not enough information for you to determine the type of citation.

Specific Numbers/Range/Exclamation

A Specific citation is annotated numbers, ranges and/or exclamation marks. For example:

Single-line: A00-1003==>P99-1029,85

Multi-line: A00-1004==>J93-2003,64-138

Multi-locations: A00-1004==>P91-1022,16!204-205
